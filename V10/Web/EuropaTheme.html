Europa theme example page

This page showcases the new web CSS styles for Flushing V10.

This large blue box represents a panel. Web pages can use panels as page headings. As you can see, the tile font is Quicksand, white both subtitle and text use Raleway. The subtitle font should be bold.

Note that the heading uses a slightly wider margin than the rest of the page. This is an acceptable divergence for web pages.

Other design remarks

Note that, while panels have their heading aligned to the left, main content headings should remain aligned to the center.

Main page content

This is the text that holds whichever content this page is supposed to contain. Under this paragraph is the web equivalent to bullet lists. Instead of using plain text, they can be replaced with cards, that present content in a flashier way. Cards can have both white and accent background.

Hyperlinks

All hyperlinks in a Flushing web page should be inside buttons (example below). This helps avoid confusion between emphasis text and navigation controls.

Cards

Note that cards have square corners, as opposed to rounded corners. This also applies to buttons and panels.

Icons

Card icons can use one of the accent colours to draw attention and convey a specific purpose. Check the intended meanings of each colour in the style guide.

Text flow

Normal text paragraphs should not have icons or other decorations, apart from headings and emphasis. Such decorations are limited to cards and panels.

Background

Cards should have a different background from the rest of the page. That can either be achieved by using another off-white colour or even pure white (#FFFFFF).

Origin

The Flushing V8 web CSS is based on MDBootstrap, and like it, is based on the Bootstrap framework.
